# 🚀 Landing Page UI — AI Job Subscription Agent

## 🎯 Goal

Build a high-converting SaaS landing page inspired by LanX, but redesigned for:

👉 AI Job Subscription Agent
(A tool that sends personalized job opportunities daily)

---

## 🎨 Design Direction (IMPORTANT)

Inspired by:

* LanX (Framer template)
* Stripe / Linear style

### Style:

* Dark theme (premium SaaS look)
* Gradient glow effects
* Glassmorphism cards
* Clean typography
* Subtle animations

---

## 🌈 Color System (IMPROVED)

Background:

* #0B0F1A (deep navy black)
* gradient overlay: radial blue glow

Primary:

* Blue: #3B82F6
* Purple: #8B5CF6

Accent Glow:

* from-blue-500/20 to-purple-500/20 blur-xl

Text:

* White: #FFFFFF
* Muted: #9CA3AF

Cards:

* bg-white/5 + backdrop-blur-lg + border-white/10

Buttons:

* gradient (blue → purple)
* hover: scale + glow

## ⚡ UI Requirements

* Smooth hover animations
* Glow effects on cards
* Responsive design
* Max width: 1200px
* Section spacing: large (py-24)

---

## 🧪 Technical Requirements

* Next.js (App Router)
* Tailwind CSS
* No UI libraries
* Component-based structure

---

## ❌ Avoid

* cluttered UI
* too many colors
* light theme (stick to premium dark)
* generic templates

---

## ✅ Success Criteria

* Looks like a real SaaS startup
* Clear value in first 5 seconds
* Strong CTA focus
* Clean + premium UI

---

## 🧠 Notes from Reference (LanX)

* Strong hero + CTA placement
* Grid-based feature sections
* Testimonials for trust
* Pricing + CTA flow

👉 But improve:

* make it more focused (less generic SaaS wording)
* more product-specific messaging

---

## 🔥 Extra Enhancements

* gradient text headline
* animated glow background
* hover lift on cards
* subtle motion on buttons

## 🎯 Final Instruction

Build a visually stunning, modern SaaS landing page that feels:
👉 premium
